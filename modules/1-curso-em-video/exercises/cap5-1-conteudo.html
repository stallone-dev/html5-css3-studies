Aula 1 - Parágrafos, caracteres e quebras

Sobre parágrafos e quebras

Para o HTML, a quebra de linha (no arquivo index.html) não tem valor semântico, portanto adicionar quebras de linha manuais resultará em um texto contínuo, uma vez que elas são ignoradas

Exemplo:

Lorem ipsum dolor sit, amet consectetur adipisicing elit. Velit odit recusandae saepe vel, reprehenderit veniam pariatur voluptates id natus possimus rerum eos excepturi rem deserunt non sunt itaque similique amet.

Para adicionar quebras de linha, é preciso indicar a tag "br" (break Row)

Exemplo:

Lorem ipsum dolor
sit, amet consectetur
adipisicing elit.
Velit odit
recusandae saepe vel,
reprehenderit veniam
pariatur voluptates
id natus possimus
rerum eos excepturi rem
deserunt non sunt itaque similique amet.

Sobre caracteres especiais

No HTML, assim como em qualquer linguagem, há caracteres reservados para fins de tokenização do texto, tornando possível o mapeamento para interpretação e compilação do código em algo útil ao computador
No caso do HTML, o principal símbolo reservado são os colchetes angulares "<" e ">" , para escrevê-los é preciso indiciar a chamada especial "& lt" e "& gt"

Um caso especial são os EMOJIs, os quais são itens da tabela UNICODE que representam símbolos quando traduzidos pelo decoder UTF-8
Para que o HTML entenda o código UNICODE diretamente, é preciso utilizar a chamada especial "&#x(CODE)" onde "(CODE)" é o código UNICODE

Exemplo:

✅ = #x2705
😁 = #x1F601
😹 = #x1F639
🐶 = #x1F436
🔥 = #x1F525
