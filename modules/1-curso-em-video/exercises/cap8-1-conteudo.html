Aula 4 - Formatações de texto

Sobre semântica no HTML5

A partir da versão 5 do HTML, foi dado ênfase à semântica na estruturação de páginas Web, visando trazer significado intrínseco ao uso da tag em si, e não apenas ao seu funcionamento
Como exemplo, a tag <b> represente "bold" ou "negrito", porém sem significado semântico, enquanto a tag <strong> traz o mesmo efeito, com significado semântico objetivo
Com o passar do tempo, tags antigas ou de significado confuso, ambíguo ou inexistente são substituídas por novas tags HTML, como o caso das tags:

  • <center>
  • <strike>
  • <font>
  • <big>
  • <applet>
  • <blink>

Sobre tags próprias para formatação

Três das principais tags usadas para formatação de texto são:

  • <strong> → Destaque de texto
  • <em> → Ênfase de texto
  • <mark> → Marcação de texto
Outras tags interessantes são:
  • <small> → Letras miúdas
  • <del> → Texto riscado/excluído
  • <ins> → Texto sublinhado/inserido
  • <sub> → Texto subscrito
  • <sup> → Texto sobrescrito
  • <abbr> → CSS é uma sigla
  • <q> → Este texto é uma citação
  • <blockquote> →
    Este texto é um bloco de citação pré-formatado
