Aula 1 - Exercícios

1. Qual dos programas a seguir é o único que não pode ser utilizado para escrever códigos em texto sem formatação para HTML + CSS + JS?

(D) - Word -> Porque o mesmo não é um editor orientado a codificação, mas sim um editor de textos para impressão


2. Qual é a empresa que fabrica o VSCode, o editor que escolhemos para escrever códigos para nosso curso?

(A) - Microsoft -> A mesma que adquiriu o GitHub e criou o Windows


3. É possível usar o VSCode em idioma Português do Brasil?

SIM -> É necessário a instalação da extensão oficial Portuguese (Brazil) Language Pack for Visual Studio Code


4. Qual é a opção que devemos habilitar nas configurações para que as instruções muito longas sejam quebradas em várias linhas automaticamente?

(D) - Word Wrap -> Essa configuração quebrará visualmente a linha para o editor, porém o arquivo em si permanecerá com a linha inteira, evitando bugs de interpretação do compilador


5. Qual é o caminho correto que devemos seguir para ligar o salvamento automático de arquivos no VSCode

(C) - Configs > Salvar Automaticamente -> Embora útil, é mais interessante criar o hábito de salvar manualmente, para quando estiver em um editor/contexto onde isso não é feito por você


6. Qual é o nome do arquivo principal de um site feito em HTML, que será aberto assim que o visitante solicitar a abertura do site?

(C) - index.html -> Este é o primeiro arquivo carregado em qualquer site da web


7. Ao criar um arquivo HTML, o VSCode tem um atalho simples para criar o código base para um arquivo desse tipo. Que atalho é esse?

(A) - ! + Enter -> Isso fará o VSCode acionar um script de configuração básica de um arquivo HTML, para que fique corretamente compatível com todos os atuais navegadores.


8. Qual é a tag HTML para criar uma linha horizontal no corpo do navegador?

(D) - hr -> É a tag semântica de "Horizontal Row"


9. Para criar um parágrafo em HTML, usamos a tag:

(A) - p -> Sendo uma das primeiras tags HTML, representa um parágrafo simples na página


10. A tag "title" é adicionada dentro de qual área de um documento HTML?

(C) - head -> O title é uma tag que modifica o título da página para o navegador--
