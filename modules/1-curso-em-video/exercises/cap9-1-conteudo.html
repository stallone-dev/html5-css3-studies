Aula 5 - Listas

Sobre listas no HTML

No HTML, é possível criar elementos similares a listas, permitindo assim a ordenação, enumeração e simplificação do processo de listagem de elementos.

Os tipos de listas são:

  • <ol> => Lista ordenada de itens
  • <ul> => Lista não-ordenada de itens
Dentro de cada um deles, é preciso declarar o item da lista através da tag:
  • <li> => Item da lista

Listas ordenadas

Listas ordenadas são listas com a características de enumeração, no HTML esses valores são automaticamente calculados conforem a quantidade de itens dentro da tag <ol>.
Exemplo:

  1. 🐕
  2. 🐈
  3. 🐖

Listas não-ordenadas

Listas não-ordenadas são usualmente usadas em itens que serão apenas pontuados na página, sem contagem / enumeração, e são feitas através da tag <ul>.
Exemplo:

  • 🐕
  • 🐈
  • 🐖

Propriedades das listas

Dentro das tags <ol> e <ul> existe uma propriedade especial chamada type, na qual pode modificar o elemento antecessor de cada item da lista.
Para o caso da lista ordenada, <ol>, os valores válidos são:

  • type = "1" => Lista numerica // PADRÃO
  • type = "a" => Lista alfabética minúscula
  • type = "A" => Lista alfabética maiúscula
  • type = "i" => Lista em algarismos romanos minúsculos
  • type = "I" => Lista em algarismos romanos maiúsculos

Para o caso da lista não-ordenada, <ul>, os valores válidos são:
  • type = "disc" => Lista em formato de esfera // PADRÃO
  • type = "circle" => Lista em formato de círculo vazio
  • type = "square" => Lista em formato de pequenos quadrados
